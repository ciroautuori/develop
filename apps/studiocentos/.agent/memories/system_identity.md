# StudioCentOS - System Identity

## Who Are We?
StudioCentOS is a **Premium Web Agency AI Platform**. We build high-performance, aesthetically stunning digital experiences.

## Core Values
1.  **Premium Quality**: Everything we build must look and feel expensive. No basic Bootstrap; use custom tailored Tailwind designs.
2.  **Gold Standard**: Our primary aesthetic is **Black & Gold**.
    - Primary: Black (`#000000`, `bg-[#0A0A0A]`)
    - Accent: Gold (`text-gold`, `border-gold`, `bg-gold/10`)
    - Font: Inter / Outfit (Modern, Clean)
3.  **Automation First**: We leverage AI agents to automate Marketing, SEO, and Content Creation.

## System Architecture
- **Frontend**: React + Vite + TailwindCSS (Monorepo `apps/frontend`)
- **Backend Services**:
    - `ai_microservice`: Python/FastAPI (Agents, RAG, Ollama)
    - `backend`: Python/FastAPI (Core Business Logic)
- **Infrastructure**: Docker Compose, Nginx Reverse Proxy.

## Key Agents
1.  **SEO Specialist**: Keyword research, Competitor Analysis (Google Search Console).
2.  **Social Media Manager**: Content generation, Posting (LinkedIn, Instagram).
3.  **Content Creator**: Blog posts, copy generation.
